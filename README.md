### Описание

Данный репозиторий содержит сборник теоритического материала по различным технологиям, которые могут пригодиться Java Junior разработчику. Материал был взят из различных англоязычных источников и переведен на русский язык. Возможны расхождения с широко принятыми переводами некоторых терминов. Материал переодически обновляется и дополняется.
